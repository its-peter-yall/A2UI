      <action>
        Create a LearningManager class with methods:
        
        - init_learning_tables(): Create learning_sessions, concept_nodes, quiz_data tables
        - create_learning_session(): Create new learning session
        - get_learning_session(): Retrieve session with node counts
        - create_concept_node(): Create node with optional quiz payload

        - learning_sessions: id, user_id, query, course_title, created_at, updated_at
        - concept_nodes: id, learning_session_id, sequence_index, title, content_markdown, status, timestamps
        - quiz_data: id, node_id, payload (JSON), created_at
