**Part B: Add difficulty gradient validation to QuizzerAgent**

In `server/agents/quizzer.py`, add a static method `_validate_difficulty_gradient()`:

```python
DIFFICULTY_ORDER = {"easy": 0, "medium": 1, "hard": 2}

@staticmethod
def _validate_difficulty_gradient(quiz_set: QuizSet) -> bool:
    """Validate that quizzes follow a non-decreasing difficulty gradient.
    
    Returns True if difficulty is ascending (or equal for adjacent quizzes).
    Returns False if difficulty is uniform across all quizzes (all same)
    or if difficulty is reversed (hard before easy).
    
    Single-quiz sets always pass validation.
    """
    if len(quiz_set.quizzes) <= 1:
        return True
    
    difficulties = [
        DIFFICULTY_ORDER.get(q.difficulty, 1)
        for q in quiz_set.quizzes
    ]
    
    # Reject uniform: all same difficulty
    if len(set(difficulties)) == 1:
        return False
    
    # Reject reversed: first quiz harder than last
    if difficulties[0] > difficulties[-1]:
        return False
    
    return True
```

Add the `DIFFICULTY_ORDER` constant at module level (after imports, before `QUIZZER_SYSTEM_PROMPT`).

**Part C: Wire validation into generate_quiz_set()**

In the `generate_quiz_set()` method (added in Plan 01), after the `self.generate()` call and after `_fix_quiz_set_option_ids()`, add validation:

```python
# Validate difficulty gradient
if not self._validate_difficulty_gradient(quiz_set):
    logger.warning(
        f"QuizSet for '{topic.title}' has invalid difficulty gradient: "
        f"{[q.difficulty for q in quiz_set.quizzes]}. "
        f"Reordering to match expected gradient."
    )
    # Reorder quizzes by difficulty (stable sort)
    quiz_set = QuizSet(
        quizzes=sorted(
            quiz_set.quizzes,
            key=lambda q: DIFFICULTY_ORDER.get(q.difficulty, 1),
        ),
        current_index=0,
    )
```

This is a soft validation — if the LLM produces wrong ordering, we fix it rather than fail. Log a warning so we can track how often this happens.

**Do NOT** add difficulty validation to the Pydantic QuizSet schema — that would break backward compatibility with existing data. Validation lives in the agent layer.
