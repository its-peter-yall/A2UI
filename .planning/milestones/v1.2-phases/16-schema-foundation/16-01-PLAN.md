3. **`test_topic_node_complexity_valid_values`** — All three valid complexity values are accepted:
   ```python
   for complexity in ["Basic", "Intermediate", "Advanced"]:
       topic = TopicNode(
           index=0, title="T", summary_for_context="S",
           key_terms=["a", "b"], complexity=complexity,
       )
       self.assertEqual(topic.complexity, complexity)
   ```

4. **`test_topic_node_complexity_invalid_rejected`** — Invalid complexity values raise ValidationError:
   ```python
   with self.assertRaises(ValidationError):
       TopicNode(
           index=0, title="T", summary_for_context="S",
           key_terms=["a", "b"], complexity="Expert",
       )
   ```

   ```python
   for count in range(1, 6):
       topic = TopicNode(
           index=0, title="T", summary_for_context="S",
           key_terms=["a", "b"], quiz_count=count,
       )
       self.assertEqual(topic.quiz_count, count)
   ```

6. **`test_topic_node_quiz_count_invalid_rejected`** — Values outside 1-5 raise ValidationError:
   ```python
   for invalid_count in [0, -1, 6, 10, 100]:
       with self.assertRaises(ValidationError):
           TopicNode(
               index=0, title="T", summary_for_context="S",
               key_terms=["a", "b"], quiz_count=invalid_count,
           )
   ```

7. **`test_topic_node_backward_compat`** — TopicNode serialized without complexity/quiz_count (simulating old data) can be constructed from dict:

   ```python
       "index": 0, "title": "Old Topic",

       "key_terms": ["term1", "term2"],
   }
   topic = TopicNode(**old_data)
   self.assertEqual(topic.complexity, "Intermediate")
   self.assertEqual(topic.quiz_count, 1)
   ```

   ```python
       summary_for_context="Deep dive into QE",
       key_terms=["entanglement", "superposition"],
       complexity="Advanced", quiz_count=5,
   )
   self.assertEqual(topic.complexity, "Advanced")
   self.assertEqual(topic.quiz_count, 5)
   ```
